## 2. 撰写总原则 · 8 条

1. **先结论，后论证**：结论放在 Block 02，位于正文第一屏。
2. **每个判断必挂事实**：形容词 = 0；事实 + 判断 ≠ 0。
3. **风险前置 + 一句话缓释**：有风险就写，别藏；同时必须给缓释方案。
4. **字段缺失留空 / 写"未披露"，不粉饰**：不再使用 `⚠️ 待补` 标签；如果整行字段未披露，**直接省略不写**；将"信息缺失"问题统一汇总到"附录 / 待确认问题清单"或"风险表"中。
5. **控制篇幅**：标准版 6000-8000 字，深度版 ≤15000 字。投委会不读作文。
6. **不打分**：技术壁垒、护城河、产品力等评估**用文字说事实和定量数据**，不用 1-5 星 / 16/25 分聚合数。
7. **报告独立性**：禁止跨标的引用（不写"相比 XX""不同于 XX 类"等表述）；每份报告只就本标的论事实、给判断。
8. **HTML 输出标签闭合校验**：`<div class="callout-strip">` / `<div class="verdict-box">` / `<div class="blueteam-box">` 等深色底块必须用 `</div>` 闭合，**严禁与 `</p>` 错配**——错配会让后续段落继承父级深色背景出现"黑字深底"不可读。生成后必须 grep 校验 `<div class=` 与 `</div>` 数量匹配。

## 3. 十个区块的撰写指引

### Block 00 · 元信息
一页纸抬头，项目名、轮次、金额、估值、我方建议投资额、日期、版本、密级。

### Block 01 · 执行摘要（Executive Summary）
- **300 字内**。
- 结构：**是什么 · 为什么现在 · 为什么我们 · 建议动作**。
- 读完这 300 字，合伙人应能立刻决定"是否继续往下读"。

### Block 02 · 推荐动作 + 核心理由 + 关键风险
- 🎯 推荐动作必须明确：投 / 跟投 / 继续跟进 / Pass。
- ✅ 核心理由 3 条：每条一句话主张 + 1-2 条支撑事实。
- ⚠️ 关键风险 3 条：每条风险 + 对应缓释方案。

### Block 03 · 公司概况
- 成立时间 / 地点 / 创始团队简介（3 行内）。
- 历轮融资表。
- 一句话业务定义（对齐技能 05 的四要素公式）。

### Block 04 · 市场与赛道
- 引用技能 01 的行研产出，**不重复论证**。
- 要点：TAM / SAM / SOM + CAGR + Why Now。

### Block 05 · 产品与技术
- 引用技能 05 的产出，重点呈现：
  - 核心价值主张
  - 产品差异化 Top 3
  - 技术壁垒评估（含 5 维度打分）
  - 技术路线图

### Block 06 · 商业模式与财务
- 收入结构 + 单位经济（CAC / LTV / 毛利率）。
- 近 12 个月关键指标走势（建议贴折线图）。
- 三年财务预测 + 合理性打分。
- 现金与 Runway。

### Block 07 · 竞争格局
- 引用技能 04 的产出。
- 市场地图 + 定位图 + 护城河评分卡。

### Block 08 · 团队
- 创始人画像：背景 / 过往成就 / 与赛道匹配度。
- 团队完整度。
- 股权结构 + 期权池。
- **关键**：背调发现（正反两面都要写，不做公关稿）。

### Block 09 · 交易条款
- 轮次、金额、估值、PS/PE 合理性。
- 关键条款逐项列出。
- 资金使用计划（必须具体到人数 / 市场预算 / 产品投入）。
- 预期退出路径。

### Block 10 · 风险与缓释
- **独立成章**，Top 5 风险。
- 风险分级：高 / 中 / 低。
- 每条必有缓释方案。
- 不允许"未来视情况调整"这种空缓释。

## 4. 语气与用词规范

### ✅ 允许
- 事实陈述："2025 Q3 ARR 达 3200 万元，同比 +180%。"
- 明确判断："毛利率 62%，显著高于同行中位数 48%。"
- 概率措辞："若 Q4 完成 A2 轮，估值大概率提升至 __ 亿元。"

### ❌ 禁止
- 形容词堆砌："前景广阔、团队优秀、技术领先" → 零信息量。
- 模糊量词："据说""大概""差不多""听起来不错" → 禁用。
- 营销话术："颠覆性、革命性、下一代、独角兽" → 除非数据支撑。

## 6. 审核三人法则

在交付前请 3 个角色自审（Agent 可分别代入）：
- 👔 **合伙人视角**：结论是否清晰、风险是否诚实？
- 🧑‍⚖️ **合规视角**：有没有漏掉法律风险？
- 🧮 **财务视角**：财务预测经得起推敲吗？
