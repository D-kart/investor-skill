# 尽职调查清单 · 模板（四维度 · Due Diligence）

**项目名称**：{{ 项目 }}
**尽调负责人**：{{ 姓名 }}
**时间窗口**：{{ 起 }} ~ {{ 止 }}
**尽调深度**：轻度 / 标准 / 深度

> 勾选约定：✅ 已核实 · ⚠️ 有疑点 · ❌ 缺失 · 🟡 进行中

---

## 一、业务尽调（Business DD）

- [ ] 产品核心功能与核心用户场景
- [ ] 真实用户 3-5 位访谈（非项目方介绍）
- [ ] 核心指标复核：DAU/MAU/NDR/GMV/ARR
- [ ] 收入结构：前 5 大客户集中度
- [ ] 获客漏斗与 CAC/LTV
- [ ] 供应链 / 关键合作方稳定性
- [ ] 近 12 个月运营数据月度趋势
- [ ] 关键合同样本（客户、供应商）

## 二、财务尽调（Financial DD）

- [ ] 近 3 年审计报告（若无则未经审计 BS/IS/CF）
- [ ] 最近 12 个月月度管理报表
- [ ] 收入口径一致性检查（合同 vs 流水 vs 账面）
- [ ] 毛利率与同行对比
- [ ] 现金 Runway 计算（月烧 vs 账上现金）
- [ ] 应收账款账龄 + 坏账准备合理性
- [ ] 关联交易与资金占用
- [ ] 未来 3 年财务预测合理性

## 三、法务尽调（Legal DD）

- [ ] 工商登记信息（股权结构、变更史）
- [ ] 股东协议、增资协议历史版本
- [ ] 知识产权归属：专利、商标、著作权、代码归属
- [ ] 未决诉讼与仲裁
- [ ] 重大合同审查：客户 / 供应商 / 劳动 / 租赁
- [ ] 员工劳动关系、期权池、社保缴纳
- [ ] 行政许可 / 资质 / 合规证书
- [ ] 数据合规：个保法 / GDPR / 跨境传输
- [ ] 境外架构（若有）：VIE / 红筹合规性

## 四、团队尽调（Team DD）

- [ ] 创始人履历真实性核查
- [ ] 创始人关键人背调：前同事 / 前老板 / 投资人
- [ ] 核心团队完整度（CXO、技术、销售、产品）
- [ ] 创始人股权 & vesting 条款
- [ ] 期权池规模（建议预留 10-15%）
- [ ] 团队分工与决策机制
- [ ] 创始人个人风险（征信、诉讼、对外投资）

---

## 输出要求

1. 每一项必须标记状态（✅/⚠️/❌/🟡）。
2. ⚠️ 与 ❌ 项必须在尽调纪要中单独说明原因与应对方案。
3. 尽调完成度 <80% 不得进入投委会。
4. 交付物：
   - 本清单（标注完成度）
   - 尽调纪要（每项 3-5 句小结）
   - 待补清单（发给项目方）
   - Red Flag 清单（直接决定 Go/No-Go）
