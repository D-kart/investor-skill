# GLOSSARY · 术语字典

> 一级市场 / 创投高频术语。Agent 在阅读材料或输出报告时遇到这些词，优先按本表释义处理。

---

## A · 主体与结构

| 术语 | 英文 | 释义 |
|---|---|---|
| LP | Limited Partner | 有限合伙人，基金出资人。 |
| GP | General Partner | 普通合伙人，基金管理人。 |
| FA | Financial Advisor | 融资财务顾问，帮项目方对接资方。 |
| SPV | Special Purpose Vehicle | 专项投资载体，常用于跟投/单项目基金。 |
| VIE | Variable Interest Entity | 可变利益实体，境外上市常用架构。 |
| 红筹 | Red Chip | 境外注册、主营在境内的架构。 |

## B · 轮次与阶段

| 术语 | 释义 |
|---|---|
| 种子轮 / Pre-Seed | 0 → 1 前，验证 idea。 |
| 天使轮 / Angel | MVP 阶段，验证 PMF 中。 |
| Pre-A / A 轮 | 初步 PMF，小规模扩张。 |
| B / C 轮 | 规模化扩张，商业模式清晰。 |
| Pre-IPO | 准备上市前的最后一轮。 |
| Bridge | 桥接轮，下一轮前的过桥。 |

## C · 条款

| 术语 | 英文 | 释义 |
|---|---|---|
| TS | Term Sheet | 投资意向书，不具法律约束力（除保密/排他）。 |
| SPA | Share Purchase Agreement | 股份购买协议。 |
| SHA | Shareholders Agreement | 股东协议。 |
| 优先清算权 | Liquidation Preference | 退出时优先获得本金 + 倍数。 |
| 反稀释 | Anti-Dilution | 下一轮估值下降时的保护条款。 |
| 回购权 | Redemption Right | 触发条件下要求公司/创始人回购。 |
| 对赌 | VAM / Valuation Adjustment Mechanism | 业绩对赌。 |
| 董事会席位 | Board Seat | 董事会观察员（Observer） / 投票席位。 |
| ROFR | Right of First Refusal | 优先受让权。 |
| Drag-Along | 拖售权 | 大股东出售时可拖带小股东。 |
| Tag-Along | 随售权 | 小股东可跟随大股东出售。 |

## D · 估值与业绩

| 术语 | 英文 | 释义 |
|---|---|---|
| Pre-Money | 投前估值 | 融资前公司估值。 |
| Post-Money | 投后估值 | Pre-Money + 融资金额。 |
| Cap Table | 股权结构表 | 历轮股东持股分布。 |
| MOIC | Multiple on Invested Capital | 投入资本倍数 = 退出金额 / 投入金额。 |
| DPI | Distributions to Paid-In | 已分配 / 已缴出资。 |
| TVPI | Total Value to Paid-In | 总价值 / 已缴出资。 |
| IRR | Internal Rate of Return | 内部收益率。 |
| ARR | Annual Recurring Revenue | 年度经常性收入。 |
| MRR | Monthly Recurring Revenue | 月度经常性收入。 |
| GMV | Gross Merchandise Volume | 商品交易总额。 |
| NDR | Net Dollar Retention | 净金额留存率。 |
| CAC / LTV | Customer Acquisition Cost / Lifetime Value | 获客成本 / 客户生命周期价值。 |
| 毛利率 / Gross Margin | — | 核心盈利能力指标。 |
| Burn Rate / Runway | — | 月烧钱速度 / 可持续月数。 |

## E · 流程

| 术语 | 释义 |
|---|---|
| 过会 / 过投决 | 过投资决策委员会（IC）。 |
| IC Memo | 投委会备忘录。 |
| DD | Due Diligence，尽职调查。 |
| KYC | Know Your Customer / Founder，了解对手方。 |
| 数据室 / Data Room | 项目方向投资方开放的资料区（VDR = Virtual Data Room）。 |
| 路演 / Pitch | 项目方向投资方展示项目。 |

## F · 市场

| 术语 | 释义 |
|---|---|
| TAM | Total Addressable Market，总可触达市场。 |
| SAM | Serviceable Available Market，可服务市场。 |
| SOM | Serviceable Obtainable Market，可获取市场。 |
| PMF | Product Market Fit，产品市场契合。 |
| NPS | Net Promoter Score，净推荐值。 |
| 护城河 / Moat | 可持续竞争优势（网络效应 / 规模 / 品牌 / 技术 / 转换成本）。 |

---

*如遇本表未收录的术语，Agent 应在输出中给出推断释义并明确标注"未经本技能术语表确认"。*
